# Enterprise Launch Checklist

## Produto

- [ ] Fluxos criticos validados ponta a ponta
- [ ] Logging estruturado habilitado
- [ ] Feature flags para modo demo vs producao
- [ ] Testes de regressao dos fluxos criticos

## Seguranca

- [ ] Secrets fora do codigo e docs
- [ ] Revisao de acesso por papel/perfil
- [ ] SCA e SAST bloqueando vulnerabilidade critica
- [ ] Politica de resposta a incidente publicada

## Operacao

- [ ] SLO/SLA definidos
- [ ] Alertas acionaveis em producao
- [ ] Backup e restore testados
- [ ] Runbook de incidente validado

## Comercial

- [ ] Mensagem comercial baseada em evidencia
- [ ] Case real com KPI financeiro
- [ ] Processo de onboarding documentado
- [ ] Proposta padrao com escopo e SLA

## Go-live

- [ ] Aprovar checklist em comite tecnico/comercial
- [ ] Executar rollout controlado
- [ ] Monitorar 72h com war room
- [ ] Registrar licoes aprendidas
